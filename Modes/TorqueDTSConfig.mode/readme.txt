TorqueDTSConfig Mode
by Danny Ngan
http://www.dannyngan.com


DESCRIPTION:

TorqueDTSConfig.mode is a SubEthaEdit syntax mode package for the Torque Game Engine's Max2DTS and Maya2DTS exporter configuration files.

Syntax coloring is provided for the following:

- Types
- Variables
- Lists
- Numbers
- Operators
- Comments
- Strings

Additional features include:

- Autocompletion of all configuration variable types
- Support for .cfg files


INSTALLATION:

Copy TorqueScript.mode into the following location:

~/Library/Application Support/SubEthaEdit/Modes

Restart SubEthaEdit if it is already open.
